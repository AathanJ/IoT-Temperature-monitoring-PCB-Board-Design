# IoT Temperature Monitoring PCB Board Design

Designed a 2-layer PCB using Altium Designer for an ESP32-based IoT Temperature and Humidity Monitoring System.

Components:
- ESP32-WROOM-32
- DHT22 Sensor
- OLED Display
- AMS1117-3.3V Regulator

Outputs:
- Schematic
- Gerber Files
- BOM
- Pick & Place Files
